## вңҰ The Problem With Every AI Tool You've Used

Every chatbot you've used вҖ” ChatGPT, Gemini, Claude вҖ” gives you **one answer, from one model, with zero accountability.**

If that model hallucinates, oversimplifies, or has a blind spot, you get a confident wrong answer with no warning.

**MindMesh AI breaks this pattern entirely.**

## вңҰ The Four Agents

| Agent | Model | Provider | What it does |
|---|---|---|---|
| рҹ’Ў **Proposer** | Llama 3.3 70B | Groq (LPU) | Generates a deep, comprehensive first answer at 300+ tokens/sec |
| рҹ—ЎпёҸ **Challenger** | Gemma 3 27B | Google AI | Hunts for logical fallacies, hallucinations, and hidden bias |
| вҡ–пёҸ **Arbitrator** | Qwen 3 235B | Cerebras | Scores both sides impartially, determines a winner with reasoning |
| вңЁ **Synthesizer** | Gemini 2.5 Flash | Google AI | Crafts the definitive final answer from the full debate history |

---

## вңҰ Key Features

- **рҹ§  Adversarial verification** вҖ” Catches errors single models miss. The Challenger is rewarded for finding problems, not agreeing.
- **вҡЎ Blazing fast inference** вҖ” Groq's LPU hardware means the Proposer answers in ~2 seconds, full debate in ~20 seconds.
- **рҹ”җ Full auth + persistence** вҖ” Clerk authentication, Neon PostgreSQL database. Your debate history is yours, forever.
- **рҹ”— Shareable debates** вҖ” Every debate gets a public link. Share your findings with anyone.
- **рҹ“ұ Installable PWA** вҖ” Works on mobile as an app. No App Store needed.
- **рҹҢҗ Production deployed** вҖ” Not a demo. Real backend on Render, real frontend on Vercel.
- **рҹ’ё 100% free to run** вҖ” Four providers, all free tiers. Zero cost for hundreds of debates per day.

## вңҰ Tech Stack

**Backend**
- **Python + FastAPI** вҖ” async API orchestration
- **Multi-provider LLM router** вҖ” Groq, Cerebras, Google AI with automatic failover
- **Pydantic** вҖ” type-safe data modeling

**Frontend**
- **Next.js 15** (App Router) вҖ” full-stack React framework
- **Tailwind CSS** вҖ” dark-first design system
- **Framer Motion** вҖ” physics-based debate animations
- **Clerk** вҖ” authentication (Google, email)
- **Neon** вҖ” serverless PostgreSQL, never pauses

**Infra**
- **Render** вҖ” Python backend hosting
- **Vercel** вҖ” Next.js frontend hosting
- **cron-job.org** вҖ” keep-alive pinger

## вңҰ Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/Soumik22-tech/MindMesh.git
cd MindMesh
```

### 2. Set up the backend

```bash
cd mindmesh
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -e .
```

### 3. Get your free API keys

| Key | Provider | Free tier | Get it |
|---|---|---|---|
| `GROQ_API_KEY` | Groq | 14,400 req/day | [console.groq.com](https://console.groq.com) |
| `GOOGLE_API_KEY_CHALLENGER` | Google AI Studio | 1,500 req/day | [aistudio.google.com](https://aistudio.google.com) |
| `GOOGLE_API_KEY_SYNTHESIZER` | Google AI Studio | 1,500 req/day | [aistudio.google.com](https://aistudio.google.com) |
| `CEREBRAS_API_KEY` | Cerebras | 1M tokens/day | [cloud.cerebras.ai](https://cloud.cerebras.ai) |

```bash
cp .env.example .env
# Fill in your keys
```

### 4. Run the backend

```bash
uvicorn mindmesh.server:app --host 0.0.0.0 --port 8000
```

### 5. Set up and run the frontend

```bash
cd ../mindmesh-web
npm install
# Add your keys to .env.local (see mindmesh-web/.env.example)
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) вҖ” ask anything.

## вңҰ Why Four Different Providers?

Most multi-agent systems use one provider for all agents. MindMesh AI deliberately uses **four different AI providers** so no single company's biases dominate the debate. Groq brings speed. Google brings breadth. Cerebras brings reasoning depth. The diversity is the feature.
